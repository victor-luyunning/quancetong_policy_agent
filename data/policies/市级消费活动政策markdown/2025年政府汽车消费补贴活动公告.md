# 2025年政府汽车消费补贴活动公告

## 一、活动时间

- **购车时间：** 2025年1月25日 0时 至 2025年3月31日 24时
  - *（以参与企业开具的机动车销售统一发票时间为准）*
- **补贴申报时间：** 2025年2月12日 10时 至 2025年4月15日 24时
- **申领资料修改时间：** 2025年2月12日 10时 至 2025年4月30日 24时

**重要提示：** 本次活动申领额度共计 **3000万元**，按照申请人提交资料的先后顺序**先到先得**，申领额度用完即止，活动可能提前结束。

## 二、活动范围

### （一）参与企业
在济南市依法注册登记的汽车销售企业，名单以活动页面公布为准。

### （二）参与对象
活动时间内，在参与企业购置**非营运乘用车新车**并完成车辆初次登记上牌的个人消费者。
- **不包括：** 非个人购车、二手车
- **不限：** 户籍和上牌地区

## 三、补贴金额与标准

对符合条件的居民个人购买乘用车发放一次性补贴。具体标准如下：

### 新能源汽车
| 购车发票金额（不含税价） | 补贴金额      |
| :----------------------- | :------------ |
| 10万元 (含) 至 30万元    | **4000元**/辆 |
| 30万元 (含) 以上         | **5000元**/辆 |

### 燃油车
| 购车发票金额（不含税价） | 补贴金额      |
| :----------------------- | :------------ |
| 10万元 (含) 至 30万元    | **3000元**/辆 |
| 30万元 (含) 以上         | **4000元**/辆 |

> **备注：**
> 1.  以上购车发票金额**不含**车辆装潢、挂牌等其它费用。
> 2.  若国家、省出台汽车报废更新或置换更新政策，符合条件的消费者可**同时申领**相关补贴（上级政策另有规定的除外）。
> 3.  **新能源乘用车定义：** 包含纯电动汽车、插电式混合动力汽车（含增程式）、燃料电池汽车三类（机动车行驶证副页有“新能源”标注，挂绿色车牌）。

## 四、申报程序及审核

### （一）申领程序
申请人完成车辆上牌，取得《机动车登记证书》和《机动车行驶证》后，即可通过以下方式申领：

**方式一：使用微信扫描活动二维码进行申报。**

![活动二维码](assets/2025年政府汽车消费补贴活动公告申报二维码.png)

**方式二：登录齐鲁银行权益平台进行申报。**
**具体路径：** 微信搜索关注“**齐鲁银行公众号**” -> 点击右下角“**领优惠**” -> “**权益中心**” -> “**济南市政府汽车消费补贴申报**”。

### （二）补贴申报资料
需在线提交以下资料的清晰原件照片：
1.  **购车人身份证**原件正反面照片（人像面和国徽面）。
2.  所购车辆的《**机动车销售统一发票**》原件照片（须为活动期内首次开具的“发票联”）。
3.  所购车辆《**行驶证**》原件主、副页照片（新能源汽车副页须有“新能源”标注）。
4.  所购车辆《**机动车登记证书**》原件第1、2页照片。
5.  购车人本人的**银行卡**照片（用于接收补贴）。
6.  申请人本人可正常接收短信的**手机号码**。

> **请确保所有图片清晰、完整，并旋转至正常可视角度，以便顺利通过审核。**

### （三）补贴审核及发放流程
1.  **审核流程：** 系统初审 -> 第三方机构复审。按提交顺序审核。
2.  **进度查询：** 消费者需及时登录平台查看审核进度，并根据提示修改资料。被退回的资料务必在 **2025年4月30日24时前** 完成修改，逾期审核不通过者不予承兑。
3.  **公示与发放：** 审核通过名单将在济南市商务局官网公示（5个工作日）。公示无异议后，补贴将发放至消费者提交的银行卡中。

## 五、注意事项

1.  **申领人责任：** 确保所填信息（特别是银行卡号）准确无误。同一申领人（以身份证、银行卡、手机号任一识别）限申领一次。发票、行驶证、登记证书、银行卡所有人信息**必须与机动车所有人一致**。对提供虚假信息骗取补贴的，将追缴资金并追究法律责任。
2.  **销售企业责任：** 严禁通过多开、虚开、伪造销售发票等手段协助骗取财政补贴。如有上述行为，将取消所涉车辆补贴资格，并追究企业责任。
3.  **活动监督：** 市商务局、齐鲁银行有权对活动进行监测，如因恶意套取补贴等行为导致财政资金面临风险，有权**提前终止活动**。
