## 山东省2025年家电以旧换新实施方案

### 一、总体要求

坚持以习近平新时代中国特色社会主义思想为指导，深入贯彻党的二十大和二十届二中、三中全会精神，落实党中央、国务院决策部署，根据省委、省政府工作要求，统筹扩大内需和深化供给侧结构性改革，统筹使用国家安排超长期特别国债支持资金，畅通家电更新消费循环堵点，全链条促进家电以旧换新，推动家电换“智”升级，持续扩大潜能消费，不断增强消费对经济发展的基础性作用。

### 二、主要任务

#### （一）继续开展家电以旧换新活动
-   **鼓励参与：** 鼓励家电生产、销售、回收、拆解等各类企业积极参与。
-   **全链条整合：** 整合上下游和线上线下资源，对换购高效家电的消费者给予补贴。
-   **便利化服务：** 推动回收网点进商圈、进便民生活圈，实现“去旧更容易，换新更愿意”。

#### （二）扩大家电以旧换新补贴范围

**1. 补贴品类（12类）：**
-   🧊 电冰箱（含冰吧、冰柜、冷柜）
-   👕 洗衣机（含干衣机、洗烘一体机）
-   📺 电视机（含激光电视、智慧屏）
-   ❄️ 空调（含家用中央空调、风管机）
-   💻 电脑（限台式电脑和笔记本电脑，其中台式电脑主机和显示器为一套）
-   🔥 热水器（含燃气壁挂炉热水器）
-   🍳 家用灶具（含家用集成灶）
-   💨 吸油烟机
-   💧 净水器（含家用净水机）
-   🍽️ 洗碗机
-   ⚡ 微波炉
-   🍚 电饭煲（含电饭锅）

**2. 补贴标准：**
| 能效/水效标准 | 补贴比例             | 备注     |
| :------------ | :------------------- | :------- |
| **2级及以上** | **15%**              | 基础补贴 |
| **1级及以上** | 15% + **5%**（额外） | 叠加补贴 |

-   **补贴上限：** 每件产品补贴不超过 **2000元**。
-   **数量限制：** 每位消费者每类产品可补贴**1件**（空调类产品最多**3件**）。
-   **政策衔接：** 2024年已享受某类家电补贴的消费者，2025年可继续享受。
-   **新增品类补贴时间：** 净水器、洗碗机、微波炉、电饭煲4类产品，补贴时间自各市细则发布之日起至**2025年12月31日**。

**3. 各市职责：** 各市自主确定上述12类产品的具体品种，并做好年度政策衔接。

#### （三）强化平台建设
-   **平台数量：** 各市**至多选择2个平台**开展活动。
-   **数据归口：** 指定**1家平台**负责与省级资格核验系统对接，统一归口数据。
-   **平台功能：** 需具备商家管理、补贴发放核销、数据分析、废旧家电回收统计等功能，提供全流程服务。

#### （四）做好资格核验
-   省商务厅建设**省级家电以旧换新资格核验系统**。
-   实时对接**中央部级系统**和**各市平台**，实现全流程闭环管理。

#### （五）营造良好市场环境
-   **诚信经营：** 引导参与主体公开承诺价格，对“先涨价后打折”等行为，取消资格并追缴资金。
-   **政策合力：** 鼓励开设补贴专区，打击假冒伪劣，支持企业联合促销，形成“组合拳”。

#### （六）完善家电售后服务体系
-   **培育领跑企业：** 推动服务进乡村、社区、企事业单位。
-   **配送时限：**
    -   家电产品自销售之日起，应在 **15日内** 完成送货安装及签收。
    -   因特殊需求（如装修）可延期，但**最长不得超过90天**，且最终签收不得晚于 **2026年1月15日**。

#### （七）加强废旧家电回收体系建设
-   培育回收试点城市和企业。
-   鼓励构建**废旧家电逆向回收体系**。
-   对规范交售废旧家电的销售企业给予支持。

### 三、保障措施

#### （一）完善组织领导
-   **方案备案：** 各市需制定工作方案，**报省商务厅备案后实施**。
-   **公平参与：** 一视同仁支持线上线下、不同所有制企业参与。
-   **目录数据库：** 建立12类家电1、2级能效/水效型号商品目录数据库。

#### （二）严格审核把关
-   **消费者信息：** 必须提供身份证明、联系方式等。
-   **企业信息：** 销售企业须提供品牌、型号、价格、补贴金额及配送签收等闭环信息；生产企业须提报能效产品清单。
-   **支付方式：** 采用**支付立减**等方式，简化流程。
-   **资金兑付：** 各市商务部门审核后，在一定周期内兑付，减轻企业垫资压力。

#### （三）加强风险防范
-   **责任主体：** 各市是资金使用的**责任主体**，负起**主体责任**。
-   **严防行为：** 严防地方保护、借机涨价、重复购买、大量囤货、骗补套补、以次充好等。

#### （四）加强补贴资金管理
| 部门               | 主要职责                                         |
| :----------------- | :----------------------------------------------- |
| **各市人民政府**   | **资金管理第一责任主体**，确保资金安全、发放及时 |
| **各市发展改革委** | 加强统筹协调，推进政策实施                       |
| **各市商务局**     | 组织实施，会同有关部门做好资金审核、兑付         |
| **各市财政局**     | 对补贴资金拨付进行监管                           |
| **各市市场监管局** | 加强产品质量和价格监管，查处违法行为             |

#### （五）开展重点核查
-   省商务厅将联合省财政厅聘请**第三方机构**进行重点核查。
-   对骗取补贴等违法行为，**追缴资金，严肃追责**，涉嫌犯罪的移送司法机关。
